# All-Notes

A self-built, topic-complete CS study repository — covering everything from C and C++ to DSA, DBMS, OS, SQL, Python, Computer Networks, and Web Development. Every note is written by hand, in detail, with code examples, visual diagrams, cheat sheets, and interview questions. Think of it as a personal textbook made public.



## What's Inside

This isn't a dump of college slides. Every subject folder is structured like a book — topics flow in order, concepts are explained clearly, and code is included with proper inline explanation. The goal was to make hard topics click fast, whether you're studying for exams, prepping for interviews, or just trying to understand something properly.

| Subject | What's Covered |
|---|---|
| **C** | Fundamentals, pointers, memory, functions, arrays, strings, file I/O |
| **C++** | OOP, classes, inheritance, polymorphism, STL, templates, exception handling |
| **DSA** | Arrays, linked lists, stacks, queues, trees, graphs, sorting, searching, recursion, DP |
| **DBMS** | ER models, normalization, transactions, ACID, concurrency, indexing, MySQL |
| **SQL / MySQL** | Queries, joins, subqueries, constraints, stored procedures, practice problems |
| **Operating Systems** | Processes, threads, scheduling, memory management, deadlocks, virtual memory, file systems |
| **Computer Networks** | OSI model, TCP/IP, routing, DNS, HTTP, subnetting, socket basics |
| **Python** | Core syntax, OOP, file handling, modules, data structures, scripting patterns |
| **Web Development** | HTML5, CSS3, JavaScript, Bootstrap, DOM manipulation, REST API basics |
| **Advance Notes** | Topics beyond the standard syllabus — deeper dives and extended concepts |



## What Makes These Notes Different

- **Written from scratch** — not copied, not summarized from Wikipedia. Every explanation is in plain language that actually makes sense.
- **Visual diagrams** — complex concepts like memory layout, process scheduling, B+ trees, and network layers have hand-made visuals to make them easier to grasp.
- **Code with explanation** — every code block has comments and context. Not just *what* the code does, but *why*.
- **Cheat sheets** — quick-reference summaries for topics you need to revisit fast before an interview or exam.
- **Interview questions** — curated questions per topic, the kind that actually show up in placements and technical rounds.
- **Topic-by-topic structure** — nothing is dumped randomly. Each subject reads like a proper chapter sequence.

## How to Use

**Browse on GitHub** — all notes are in `.md` format and render cleanly in the browser. Just navigate to the subject folder and read.

**Clone locally** for offline access or to open in VS Code with a Markdown preview:

```bash
git clone https://github.com/nishant05gaurav/All-Notes.git
cd All-Notes
```

Open any `.md` file in VS Code and press `Ctrl+Shift+V` (or `Cmd+Shift+V` on Mac) to render the preview alongside the source.
